# Simple Python Example Project

This workspace now contains a small Python project to help you learn:

- `hello_world.py` - a simple program that prints a greeting.
- `calculator.py` - reusable functions for basic math operations.
- `main.py` - a small command-line program that uses `calculator.py`.

## How to run

1. Open this folder in VS Code.
2. Open `main.py`.
3. Run it by using the Run button in VS Code or by opening a terminal and running:
   ```bash
   python main.py
   ```

## What to learn

- `print()` shows text in the terminal.
- `def` creates a function.
- `import` brings code from another file into your program.
- `if __name__ == "__main__":` makes `main.py` run only when executed directly.
- You can edit the messages and math logic to see how it changes.
